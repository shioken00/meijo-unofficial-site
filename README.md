# 名城大学情報工学部 非公式HP

GitHub Pagesで公開できる静的Webサイトです。

## 公開方法

1. GitHubで新しいリポジトリを作成
2. このフォルダの中身をすべてアップロード
3. Settings → Pages → Deploy from a branch
4. Branchを `main`、Folderを `/(root)` に設定
5. 数分後に `https://ユーザー名.github.io/リポジトリ名/` で公開
